# Markdown Cheat Sheet

This is a quick reference guide to common Markdown syntax.

## Headings

# H1
## H2
### H3
#### H4
##### H5
###### H6

## Emphasis

*Italic* or _Italic_

**Bold** or __Bold__

**_Bold and Italic_**

~~Strikethrough~~

## Lists

### Unordered List

- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2
- Item 3

### Ordered List

1. First item
2. Second item
   1. Subitem 2.1
   2. Subitem 2.2
3. Third item
